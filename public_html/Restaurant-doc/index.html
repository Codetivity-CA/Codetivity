Getting Started

Unzip the downloaded folder and find out the Resturant-Onepage-HTML5-Business-Template-master folder. There are all the template files in this folder. You can open the following files for viewing in browser.

  1. /css — folder with css files.
  2. /fonts — folder with icon fonts.
  3. /img — folder with image files.
  4. /js — folder with Javascript files.
  5. index.html — the main template file.
  6. process.php — this is for the contact form section.

Upload the template files (template folder) to server with the help of one of the FTP-clients (for example, filezilla).
